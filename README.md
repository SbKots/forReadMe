# FileManagerApp

**FileManagerApp** — консольное приложение на **C#**, предназначенное для работы с пользовательскими данными и файлами разных форматов.

> Проект позволяет читать, редактировать, сортировать, искать и сохранять данные пользователей в нескольких форматах.

## Что реализовано

### Основные возможности

- чтение данных из файла
- запись данных в файл
- вывод данных на экран
- поиск пользователя по имени
- сортировка по имени
- сортировка по возрасту
- добавление новой записи
- удаление записи
- изменение записи

### Пошаговая работа программы

1. Пользователь выбирает нужное действие.
2. Программа получает данные.
3. Выполняется обработка списка пользователей.
4. Результат выводится на экран или сохраняется в файл.

## Структура проекта

## FileManagerApp

Основное консольное приложение, содержащее:

- меню программы
- обработку ввода пользователя
- вызов методов библиотеки

### FileManagerLib

Библиотека классов, в которой находятся:

1. модель данных `User`
2. интерфейс `IFileHandler<T>`
3. обработчики файлов
4. логика управления пользователями

### FileManagerTest

Тестовый проект, который содержит unit-тесты для проверки:

- чтения файлов
- записи файлов
- сортировки
- поиска
- редактирования данных

## Пример кода

Ниже показан пример интерфейса для работы с файлами:

```csharp
public interface IFileHandler<T>
{
    List<T> Read(string filename);
    void Write(string filename, List<T> data);
}
```

Пример модели пользователя:

```csharp
public class User
{
    public int Id { get; set; }
    public string Name { get; set; }
    public int Age { get; set; }
    public string Role { get; set; }
    public DateTime RegisterDate { get; set; }
    public Aktivnost Activity { get; set; }

    public enum Aktivnost
    {
        Онлайн,
        Оффлайн
    }
}
```
